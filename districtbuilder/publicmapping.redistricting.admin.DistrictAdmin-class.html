Class DistrictAdmin

source code

Administrative setting for Districts.

Districts are composite sets of geometry that are built out of Geounits. Each district aggregates the Characteristics of the Geounits that are contained within in, and stores those results in ComputedCharacteristic models.

This class controls the display of an OpenStreetMap slippy map for the geometry fields in the District model.
