Class Circle

source code

Helper class for the Roeck calculator. A Circle class can create circles based on 1, 2, or 3 coordinates. A circle has a center and a radius. A circle also can test if another point lies within the area of itself.

__init__(self, pts)

source code 

Create a new Circle helper. Constructing a Circle with one coordinate creates a 0 diameter circle, centered on the coordinate. Constructing a Circle with two coordinates creates a circle positioned between the two coordinates, with the diameter set to the distance between the points. Constructing a Circle with three coordinates creates a circle that passes through all three points.

Parameters:
  • pts - A set of points that lay on the perimeter of this circle.
Returns:
A calculated Circle that contains the coordinates.

deperpendicularize(self, pts)

source code 

Reorder coordinates to ensure that they are not perpendicular in a way that may cause divide by zero exceptions. This tests all 6 permutations of the point set, testing it with the isperpendicular method.

Returns:
A 3 element tuple with the reordered points.

isperpendicular(self, pt1, pt2, pt3)

source code 

Test a set of points to see if they are perpendicular. Points are deemed perpendicular if they describe two different lines that are parallel to the X and Y axes.

Parameters:
  • pt1 - The first point to test.
  • pt2 - The second point to test.
  • pt3 - The third point to test.
Returns:
A boolean flag indicating if the points describe a set of perpendicular lines that are parallel with the X and Y axes.
