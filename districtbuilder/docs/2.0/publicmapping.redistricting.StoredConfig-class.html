Class StoredConfig

source code

A class that is represented on disk by a configuration file. This class provides methods of reading the configuration file.

An example configuration file for DistrictBuilder is in the github repository: https://github.com/PublicMapping/DistrictBuilder/blob/master/docs/config.dist.xml
