Class ComputedPlanScore

source code

A score generated by a score function for a plan that can be saved for later.

These computed scores contain version numbers, since a plan's version number is incremented each time, but scores need to exist for different plan version numbers, for history, etc.

Get the computed value. This method will leverage the cache when

If the cached score exists, its value is not changed.
