check_and_update(a_model, unique_id_field='name', overwrite=False, **kwargs)

source code 

Check whether an object exists with the given name in the database. If the object exists and "overwrite" is True, overwrite the object. If "overwrite" is false, don't overwrite. If the object doesn't exist, it is always created.

Parameters:
  • a_model - A model class.
  • unique_id_field - The field name that is unique.
  • overwrite - Should the new object be overwritten?
Returns:
tuple - the object in the DB after any changes are made, whether the object had to be created, whether the given attributes were consistent with what was in the database, a message to return indicating any changes.
