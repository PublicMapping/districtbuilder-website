Module Hierarchy

  • publicmapping: Define publicmapping as a Python package.
    • publicmapping.context_processors: Context processors for the DistrictBuilder web application.
    • publicmapping.redistricting: Redistricting django application and supporting classes.
      • publicmapping.redistricting.admin: Configure the appearance of the redistricting admin interface.
      • publicmapping.redistricting.calculators: Define the calculators used for scoring in the redistricting app.
      • publicmapping.redistricting.config: Configuration module for DistrictBuilder
      • publicmapping.redistricting.forms: Define the forms used by the redistricting app.
      • publicmapping.redistricting.management: Define redistricting.management as a Python package.
        • publicmapping.redistricting.management.commands: Define redistricting.management.commands as a Python package.
          • publicmapping.redistricting.management.commands.database_i18n: Install the internationalization messages for a running DistrictBuilder instance.
          • publicmapping.redistricting.management.commands.exportplans: A django management command to export plans.
          • publicmapping.redistricting.management.commands.makelanguagefiles: Collect internalionalization messages and compile the message files for use by the gettext library.
          • publicmapping.redistricting.management.commands.numusers: Count the number of users of the DistrictBuilder application.
          • publicmapping.redistricting.management.commands.reaggregate: Reaggregate one or more plans in the DistrictBuilder web application.
          • publicmapping.redistricting.management.commands.removescoreconfig: Delete score configurations from the DistrictBuilder application.
          • publicmapping.redistricting.management.commands.setup: Set up DistrictBuilder.
      • publicmapping.redistricting.models: Define the models used by the redistricting app.
      • publicmapping.redistricting.reportcalculators: Define the calculators used for calculator reports in the redistricting app.
      • publicmapping.redistricting.tasks: Utilities for the redistricting application: format conversion, useful database queries, etc.
      • publicmapping.redistricting.templatetags: Define the custom templatetags as a Python package.
        • publicmapping.redistricting.templatetags.redistricting_extras: Extra tags and templates used in the django templating system for the redistricting app.
      • publicmapping.redistricting.tests: Define a set of tests for the redistricting app.
      • publicmapping.redistricting.views: Django views used by the redistricting application.
    • publicmapping.setup: Set up DistrictBuilder.
    • publicmapping.views: Define project views for this django project.
