Class Sum

source code

Sum up all values.

For districts, this calculator will sum up a series of arguments.

For plans, this calculator will sum up a series of arguments across all districts. If a literal value is included in a plan calculation, that literal value is combined with the subject value for each district.

For lists of numbers, this calculator will return the sum of the list.

Each argument should be assigned the argument name "valueN", where N is a positive integer. The summation will add all arguments, starting at position 1, until an argument is not found.
