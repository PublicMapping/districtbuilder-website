Class SubjectAdmin

source code

Administrative setting for Subjects.

Subjects are the names of attributes of Geounits. Each Geounit has one Characteristic per Subject.

get_urls(modeladmin)

source code 

Get the URLs for the auto-discovered admin pages. The Subject admin pages support a GET url for a subject template, an upload endpoint, and an upload/uuid/status endpoint to check the status of an upload.

has_delete_permission(modeladmin, request, obj=None)

source code 

Override the delete permission for the Subject model. A Subject may never be deleted. This only affects the admin change form, and not the bulk delete dropdown in the Subject list screen.
